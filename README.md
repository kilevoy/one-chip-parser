# Парсер one-chip.ru

Собирает товары категорий **Загрузчики** и **Редакторы** (со всеми подкатегориями) с сайта https://one-chip.ru.

Для каждого товара сохраняются:
- название
- ссылка на изображение
- описание (текст и HTML)
- цена (и валюта), либо «Цена по запросу» в поле `price_text`
- вариации (sku-варианты с ценой)
- характеристики
- артикул, бренд, наличие, путь по категориям, корневая категория
  (`discovered_root` — Загрузчики / Редакторы) и точная подкатегория, в которой
  найден товар (`discovered_in`)

Результаты сохраняются рядом с приложением (или в выбранную папку):
- `products.json`
- `products.csv` (UTF-8 BOM, разделитель `;`, открывается в Excel)
- `products.xlsx`

## Запуск из готового exe

Если у вас есть собранный `one-chip-parser.exe` — просто запустите его двойным кликом. Python не нужен.

## Запуск из исходников

Нужен Python 3.10+ (`tkinter` идёт в комплекте с Python для Windows).

```cmd
pip install requests beautifulsoup4 lxml openpyxl
```

Двойной клик по `run.bat` или из консоли:

```cmd
python app.py
```

## Сборка одного exe для заказчика

Двойной клик по `build_exe.bat`. По окончании готовый файл лежит в `dist\one-chip-parser.exe`. Передавайте только этот файл — никаких дополнительных установок не требуется.

## Что показывает окно

- отметьте нужные категории (по умолчанию обе);
- укажите количество потоков (8 — оптимально);
- выберите папку для результатов;
- нажмите **▶ Запустить**.

В логе видны все шаги: сбор подкатегорий → ссылки на товары → парсинг карточек → сохранение. Прогресс-бар отражает обработку карточек. Кнопкой **■ Остановить** парсинг можно прервать в любой момент.

## Запуск из консоли (без GUI)

```cmd
python parser.py --threads 8 --out-dir results
```

Опции:
- `--threads` — количество потоков (по умолчанию 8);
- `--out-dir` — папка результатов (по умолчанию текущая);
- `--only zagruzchiki|redaktory` — только одна корневая категория.
